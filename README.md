# Ai-Agent
1.Accepts a coding task (e.g., "strings aren't splitting in my app, plz fix 🥺👉🏽👈🏽")

2.Chooses from a set of predefined functions to work on the task, for example:

    A.Scan the files in a directory

    B.Read a file's contents

    C.Overwrite a file's contents

    D.Execute the Python interpreter on a file

3.Repeats step 2 until the task is complete (or it fails miserably, which is possible)
